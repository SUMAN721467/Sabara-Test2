# Vercel Deployment

Use `crafted-canvas` as the Vercel root directory.

## Build Settings

- Install command: `npm install`
- Build command: `npm run build`
- Output directory: `dist/client`

These are also set in `vercel.json`.

## Required Environment Variables

Set these in Vercel Project Settings > Environment Variables:

- `SUPABASE_URL`
- `SUPABASE_PUBLISHABLE_KEY`
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_PUBLISHABLE_KEY`
- `ADMIN_EMAILS`
- `VITE_ADMIN_EMAILS`

Optional, depending on features:

- `SUPABASE_SERVICE_ROLE_KEY` for full admin customer/profile reads
- `RESEND_API_KEY` and `RESEND_FROM_EMAIL` for order confirmation emails
- `SITE_URL` for absolute links in emails

## Notes

- `api/index.js` runs as a Vercel Edge Function and forwards requests to the TanStack Start server build at `dist/server/server.js`.
- `dist` is generated by the build and should not be committed.
